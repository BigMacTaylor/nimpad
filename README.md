# Nimpad

A very simple text editor with theme support and syntax highlighting.
Written in Nim, you can use it as is, or as a building block for your own IDE.


## Install

### Prerequisites
- Nim 2.2.4
- Gtk3
```bash
nimble install https://github.com/BigMacTaylor/nimpad.git
```

## Keybindings

| Key | Action |
|-----|--------|
| `Ctrl`, `s` | Save |
| `Ctrl`, `Shift`, `s` | Save As |
| `Ctrl`, `z` | Undo |
| `Ctrl`, `Shift`, `z` | Redo |
| `Ctrl`, `x` | Cut |
| `Ctrl`, `c` | Copy |
| `Ctrl`, `v` | Paste |
| `Ctrl`, `a` | Select All |
| `Ctrl`, `f` | Find |
| `Ctrl`, `g` | Find Next |
| `Ctrl`, `Shift`, `g` | Find Previous |
| `Ctrl`, `r` | Replace |
| `Ctrl`, `q` | Quit |
